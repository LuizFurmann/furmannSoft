Aplicativo de tarefas com Calendário

Neste aplicativo utilizei a linguagem Kotlin, banco de dados Room Database e arquitetura MVVM, com este aplicativo o usuário consegue criar tarefas e definir uma rotina com frequências, como, semanal, quinzenal, etc.


Tela home onde mostra as tarefas do dia atual

Tela com um calendário onde o usuário pode ter uma visão de próximas tarefas, clicando no dia escolhido, será mostrado as tarefas para este dia

Tela do mês atual mostrando as tarefas para o dia selecionado

Tela para criação de uma nova tarefa

Tela de navegação com opções de frequências de tarefas a serem mostradas

Tela de edição de uma tarefa, mostrando os dados da tarefa selecionada para ser editada
